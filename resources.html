The GDScript Language


What is GDScript?

According to the Godot Documentation

"GDScript is a high-level, object-oriented, imperative, and gradually typed programming language built for Godot. It uses an indentation-based syntax similar to languages like Python. Its goal is to be optimized for and tightly integrated with Godot Engine, allowing great flexibility for content creation and integration."

GDScript is entirely independent from Python and is not based on it.

Why consider GDScript?


A Powerful Language

GDScript is a high-level, dynamically typed scripting language specifically designed for Redot Engine. It offers several advantages that make it a powerful tool for game development

not to mention that its way faster to use when using the Redot Engine

Easy to Learn

GDScript, the scripting language specifically designed for Godot Engine, is renowned for its simplicity and readability

When you look at the typing, it looks very similar to Python, and due to it being a high-level programming language, it is very easy to comprehend the language, giving time on developing more.
